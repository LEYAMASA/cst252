Problems

In this lab, we learned on how to do basic debugging which is the method of finding dysfunctional code functions within our code. there are two different kinds of errors that exist within using our browser debugging console: syntax and logical errors. The former is when the system doesn't understand any manual command inputs and the latter is when your code is correct, but doesn't do what you intended to do.

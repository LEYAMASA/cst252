Lab 14 Loops

Problems

In this lab, we learned on how to do loop functions within JS code. Loops allows the performing of the same tasks over and over again within JS coding. We're introduced of two kinds of loops: "For"Loops and "While" Loops as the former determines you knowing how many set number of times you want to repeat tasks as the latter is for when you don't know how many times you need to repeat the task on the code.

Problems

In this lab, we continue to use jQuery and learn about Expressions and Conditionals of a code program. Expressions are to test with booleans to verify something as either true or false and Conditionals are programming code that checks under different conditions to activate when certain values are typed in response.

Challenges

This lab combines the challenges of both Expressions And Conditionals as the lab assigned to use the mod function to assign numerical expressions for each of the four Harry Potter houses as well as creating else if statements applied onto each of the four houses. These two tied together with both input and button functions tied along with the SortingHat variables and a large plethora of text for the four houses.

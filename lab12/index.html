Lab 12 AdvancedjQuery

Problems

In this lab, we explore more into using jQuery as we learn more coding techniques within JS such as modifying classes/ids and input elements. It involves of changing CSS properties without the need of JS commands and keep CSS properties intact as well as using JS to respond to the values entered in a form.

Challenges

This lab wasn't too hard to accomplish as the main challenge of this lab is to toggle around button elements to react according to the styles put into CSS properties. It really emphasizes the changing of static values into a different property by jQuery itself. This was an overall interesting lab experience.

Problems

The former is derived from our last lab's focus on object prototypes and the latter is for laying out a general approach in programming JavaScript Classes. Classes are more organized than of using standard prototype structuring. On the other hand OOP is an outline of steps to code in JS Classes providing details of implementing the JS class code.
